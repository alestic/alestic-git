

                          Alestic Git Server


This software is used to build the Alestic Git Server AMIs for use
with Amazon EC2.

Information about this project is available on the main Alestic Git
Server web page:

                   http://alestic.com/alestic-git/

The software used to build the AMIs is published here for transparency
and in the event that others wish to build their own AMIs following
the same process.
